# anti-rkn
# Новый удобный способ разблокировки Discord!

**1.** Скачиваем архив "anti-rkn.zip"

**2.** Разархивируем в любую удобную папку

**3.** Запускаем  файл anti-rkn

**4.** Далее появиться 2 предупреждающих окна от Windows, нажимаем подробнее и кнопку "Выполнить в любом случае" (Мы предоставляем доступ к контролю сети (скоростного соединения))

**6.** Далее в открытом окне нажимаем кнопку "Запустить" и перезагружаем Discord! 

Данное программное решение было сделано на основе работы GoodbyePDI

**Телеграм с обновлениями https://t.me/discorddpi**
